# AI-Powered Study Planner (Full-Stack Application)

## Overview

The AI-Powered Study Planner is a full-stack application designed to help students efficiently manage their academic tasks and optimize study schedules. The system integrates a modern frontend, a scalable backend, and AI-driven logic to provide intelligent task recommendations and improve productivity.

---

## Key Features

* Task creation, update, and deletion
* Intelligent study recommendations using AI logic
* Structured task management with backend API
* Full-stack integration (Frontend + Backend + AI modules)
* Scalable and modular architecture

---

## Technology Stack

### Frontend

* React (Vite)
* JavaScript
* HTML5, CSS3

### Backend

* Python
* Flask (REST API)
* SQLite

### AI / Logic

* Custom suggestion engine
* Rule-based / intelligent task prioritization

---

## Project Architecture

The application follows a modular full-stack architecture:

* **Frontend**: Handles user interface and interactions
* **Backend**: Manages APIs, business logic, and data storage
* **AI Modules**: Provide intelligent recommendations for study planning

## Installation and Setup

### 1. Clone the Repository

git clone https://github.com/your-username/ai-study-planner.git
cd ai-study-planner

---

### 2. Backend Setup

cd backend
pip install -r requirements.txt
python app.py

---

### 3. Frontend Setup

cd frontend
npm install
npm run dev

---

## API Endpoints

* `/tasks` → Task management operations
* `/ai/suggestions` → AI-based study recommendations
